Week 15: Patriots vs Bengals


Patriots run defense vs Bengals running game

The Bengals have a bottom 5 running game in 2019. Their offensive line has struggled to get off their blocks, and thus failing to create lanes for their running backs.

The Patriots, on the other hand, have a top 5 ranked run defense. They've looked very good on that front over the last two weeks, but they've been stout all season.

Match-up edge: Patriots.


Patriots pass rush vs Bengals pass protection

With one of the premier pass rushing units in the league, the Patriots should be able to exploit the Bengals' offensive line, and in the process, disrupt their passing attack.

Match-up edge: Patriots.


Patriots pass defense vs Bengals passing game

This will be a matchup between the best secondary in the league and a bottom half passing attack. The Patriots have shut down some of the best WRs and pass offenses in the league since last season, and should continue to see success on Sunday

Match-up edge: Patriots.


Patriots running game vs Bengals run defense

The Patriots ground game has had flashes of brilliance occasionally, but has been subpar overall in 2019. This is due to an injury depleted offensive line and some disappointing performances from 2nd year RB Sony Michel. The offensive line continues to get healthy, with the return of Ted Karras, and there is no better game for them to get their ground game going, as they will be up against the 32nd ranked run defense.

Match-up edge: Patriots.


Patriots pass protection vs Bengals pass rush

Ted Karras' return is much needed, as the Chiefs steamrolled the interior of the Patriots OL last week. The Bengals have a solid pass rush group, but the Patriots OL are poised for a bounce back week.

Match-up edge: Patriots.


Patriots passing game vs Bengals pass defense

Another struggling position group for the Patriots, the receivers are getting healthier, and they've got a favorable matchup this week. They should get James White, Mohamed Sanu, and N'Keal Harry more involved this week, as they look to build momentum towards the playoffs.

Match-up edge: Patriots.
